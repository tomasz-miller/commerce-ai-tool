# Contributing

Thank you for your interest in contributing to Commerce AI Tool!

## Development setup

1. Fork and clone the repository
2. Install dependencies: `pnpm install`
3. Copy `apps/demo-next/.env.example` to `apps/demo-next/.env.local`
4. Build packages: `pnpm build`
5. Run demo: `pnpm dev`

## Code style

- TypeScript strict mode
- English for code comments and documentation
- Run `pnpm lint` and `pnpm typecheck` before submitting PRs

## Pull requests

1. Create a feature branch from `main`
2. Keep changes focused and minimal
3. Update documentation if APIs change
4. Ensure CI passes

## Reporting issues

Use GitHub Issues with a clear description, steps to reproduce, and environment details.
